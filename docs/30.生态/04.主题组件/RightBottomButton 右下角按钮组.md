右下角按钮组组件提供返回顶部、跳转到评论区功能。

## 基础使用

```ts
import DefaultTheme from "vitepress/theme";
import { TkRightBottomButton } from "vitepress-theme-teek";
import { h } from "vue";

export default {
  extends: DefaultTheme,
  Layout: () => h("div", null, [h(TkRightBottomButton), h(DefaultTheme.Layout)]),
};
```
