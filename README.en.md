## Local Development

Project cloning

```bash
git clone https://github.com/Kele-Bingtang/vitepress-theme-teek.git
```

Dependencies installation (only pnpm can be used to install dependencies)

```bash
pnpm install
```

Build referenced packages

```bash
pnpm stub
```

Start documentation project

```bash
pnpm docs:dev
```
